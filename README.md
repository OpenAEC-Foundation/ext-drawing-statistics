# Drawing Statistics

Sample extension for Open 2D Studio that displays shape counts and drawing statistics.

## Features

- Adds an **Add-Ins** ribbon tab with a **Shape Count** button
- Registers a terminal command: `stats show`
- Counts all entities by type in the current drawing

## Installation

Add this extension to your Open 2D Studio extensions directory.

Requires Open 2D Studio v0.15.0 or later.
